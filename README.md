#  TradeAI — Торговая платформа с AI-прогнозированием

![Django](https://img.shields.io/badge/Django-6.0-green)
![Python](https://img.shields.io/badge/Python-3.13-blue)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-purple)
![License](https://img.shields.io/badge/License-MIT-yellow)

TradeAI — это современная веб-платформа для торговли на фондовом и валютном рынках с элементами искусственного интеллекта для прогнозирования цен.


##  Основные возможности

###  Аутентификация и безопасность
- Регистрация и авторизация пользователей
- Кастомная модель пользователя с расширенными полями
- Разграничение доступа к разделам

###  Торговые инструменты
- Просмотр доступных активов (акции, валюты, криптовалюты)
- Фильтрация по категориям
- Детальная информация о каждом активе
- Покупка активов в портфель

###  Аналитика и прогнозы
- Интерактивные графики цен (Matplotlib)
- AI-прогнозирование движения цены
- Выбор временных периодов (1Д, 1Н, 1М, 3М, 6М, 1Г)
- Экспорт отчётов в CSV

###  Управление портфелем
- Отслеживание баланса и активов
- История операций
- Расчёт доходности

###  Настройка торговли
- Выбор маржи (изолированная/кросс)
- Установка кредитного плеча
- Настройка торговой валюты
- Управление процентом депозита

###  Интеграция с биржами
- Подключение через API-ключи
- Поддержка популярных биржевых платформ

###  Новости рынка
- Финансовые новости по категориям
- Важные/главные новости
- Детальные страницы с полным текстом

###  Пополнение счёта
- Несколько способов оплаты
- История транзакций
- Безопасные платежи

##  Технологический стек

| Категория | Технологии |
|-----------|------------|
| **Backend** | Django 6.0, Python 3.13 |
| **База данных** | PostgreSQL / SQLite |
| **Frontend** | Bootstrap 5, HTML5, CSS3, JavaScript |
| **AI/ML** | NumPy, Pandas, Matplotlib, TensorFlow/Keras |
| **Данные** | Yahoo Finance API, Dukascopy |
| **Деплой** | Gunicorn, Nginx (опционально) |
